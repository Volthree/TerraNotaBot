# TerraNotaBot_V1.0.1

This is TerraNotaBot. <br />
You can create geographic map image depends on: <br />

Map size -  for picture size <br />
Scale -  for pixel-style  <br />
Height difference -  for heights <br />
Islands modifier -  for land area <br />
